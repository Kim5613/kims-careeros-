**记忆工作流：**
```
对话中 → AI 判断值得记 → saveMemory 入库
下次对话 → buildContext 检索匹配记忆 → 注入"你对 Kim 的了解"
芝士: "你上次提到的字节HRBP有回复了吗？" ← 自然衔接
```

**记 vs 不记规则（AI 自动判断）：**
- ✅ 记：偏好、个人信息、操作、决策（下次用得上的）
- ❌ 不记：一次性查询、寒暄、知识问答、重复信息
